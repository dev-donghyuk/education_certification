[처음으로 배우는 자바스크립트]

자바스크립트는 html이 가지고 있지 않은 기능들을 보완하기 위해 만들어진 프로그래밍언어입니다. 주로 html문서내에 요소들을 조작(수정,추가,제거 등)을 하기 위해 만들어진 언어라
고 할 수 있습니다.

[자바스크립트로 할 수 있는 일들]

  1. HTML의 내용을 변경할 수 있다.
  2. HTML의 속성을 변경할 수 있다.
  3. HTML의 스타일을 변경할 수 있다.
  4. HTML의 요소를 추가하거나 제거할 수 있다.
  5. HTML 요소를 보이거나 감출 수 있다.

[자바스크립트 선언방식 3가지]

  1. 태그에 직접 기술
  2. 스크립트 태그 사이에 기술
  3. 외부 파일로 만든 후 호출하여 사용

[자바스크립트 선언 순서]

  1. 대상 지정하기
  2. 이벤트 발생하기
  3. 실행할 처리내용 선언
